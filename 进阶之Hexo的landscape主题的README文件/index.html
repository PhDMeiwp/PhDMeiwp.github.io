本文作者：导航栏logo字体的设置，即更改个人网站title和subtitle字体的颜色：

找到如下路径的header.styl文件，

E:~\Blog\Hexo-cn\themes\landscape\source\css_partial\header.styl

将原始代码：

$logo-text
 text-decoration: none
 color: #fff    #白色字体
 font-weight: 300
 text-shadow: 0 1px 4px rgba(0, 0, 0, 0.3)

改为如下代码即可，颜色为css代码——可参考 http://flatuicolors.com/ 网站：

$logo-text
 text-decoration: none
 color: #3498db    #蓝色字体
 font-weight: 300
 text-shadow: 0 1px 4px rgba(0, 0, 0, 0.3)
